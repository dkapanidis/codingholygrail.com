# DEVELOP

## Run locally

Site can be previewed locally with [nextjs].

[nextjs]: https://nextjs.org/

* To install:

```shell
npm install
```

* To start:

```shell
npm run dev
```

## Deployment

Continuous Deployment (CD) is done with [Netlify]:

* from `master` branch to https://codingholygrail.com/.

[Netlify]: https://www.netlify.com/
